<h1>💬 iChat App – Simple Realtime Chat Interface</h1>
<h3>📌 Description</h3>

iChat App is a simple web-based chat interface inspired by modern messaging platforms.<br>
This project is built using HTML, CSS, and JavaScript to demonstrate the basic structure and user interface of a realtime chat application.<br>

The application allows users to type and send messages in a clean chat layout.<br> It is designed as a practice project for frontend development and realtime communication concepts.

This project is ideal for beginners who want to understand how chat applications are structured on the frontend before integrating backend technologies like Node.js and Socket.io.
<hr>
<h3>🚀 Features</h3>
<li>Clean and simple chat interface</li>
<li>Message input and send button</li>
<li>Chat message container layout</li>
<li>Custom logo support</li>
<li>Responsive layout structure</li>
<li>Ready for realtime integration with Socket.io</li>
<hr>
<h3>🛠️ Technologies Used</h3>
<li>HTML5 – Structure of the application</li>
<li>CSS3 – Styling and layout design</li>
<li>JavaScript (Vanilla JS) – Client-side functionality</li>
